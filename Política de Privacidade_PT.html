Política de Privacidade dos aplicativos

Este aplicativo utiliza diversos serviços para exibição de anúncios, análise de uso e garantia da funcionalidade técnica. O tratamento de dados pessoais é realizado de acordo com as leis de proteção de dados aplicáveis, especialmente o Regulamento Geral sobre a Proteção de Dados (GDPR) no Espaço Econômico Europeu, a Lei de Privacidade do Consumidor da Califórnia (CCPA) e a Lei de Direitos de Privacidade da Califórnia (CPRA) nos Estados Unidos, bem como a Lei Geral de Proteção de Dados (LGPD) no Brasil.

2. Tratamento de Dados no Aplicativo

Durante o uso deste aplicativo, os seguintes dados pessoais podem ser coletados e processados:

  • Localização aproximada, usada para publicidade baseada em localização e fins analíticos.
  • ID do dispositivo (por exemplo, ID de publicidade ou IDFA), usada para análises e publicidade.
  • Interação com o aplicativo (por exemplo, cliques, navegação no menu, progresso no jogo), usada para otimizar a funcionalidade e o desempenho publicitário.
  • Dados de anúncios (por exemplo, quais anúncios foram exibidos e se foram clicados).
  • Dados de falhas coletados para identificar e corrigir erros.
  • Dados de desempenho, como tempos de carregamento ou uso de bateria, usados para otimização técnica.

Parte desses dados pode estar vinculada ao ID do dispositivo e ser usada para fins de rastreamento, especialmente em conexão com publicidade via Google AdMob.

3. Uso do Google AdMob

Este aplicativo utiliza o Google AdMob, um serviço da Google Ireland Limited, para exibir anúncios. A Google processa dados sobre o dispositivo, localização aproximada, uso do aplicativo e interações com anúncios. Esses dados podem ser usados para exibir anúncios personalizados ou analisar o desempenho da publicidade.

O aplicativo transmite automaticamente sinais de privacidade apropriados (por exemplo, Restricted Data Processing ou IAB Privacy Framework) para cumprir com os requisitos legais.

Mais informações sobre o tratamento de dados pela Google podem ser encontradas em:
https://policies.google.com/privacy

Informações para Usuários da União Europeia (GDPR)

Antes do tratamento de dados pessoais para fins publicitários, é solicitada uma autorização através do Google Consent Framework (User Messaging Platform). Em dispositivos iOS, é apresentada adicionalmente uma solicitação via App Tracking Transparency (ATT) da Apple.

O tratamento dos dados se baseia nos seguintes fundamentos legais:

  • Artigo 6(1)(a) do GDPR (consentimento)
  • Artigo 6(1)(f) do GDPR (interesse legítimo, por exemplo, segurança ou correção de erros)

Quando os dados forem transferidos para países terceiros, especialmente os EUA, isso será feito com garantias adequadas (por exemplo, certificação da Google sob o EU-U.S. Data Privacy Framework).

Os usuários têm direito de acessar, corrigir, apagar, restringir ou se opor ao tratamento de seus dados, bem como direito à portabilidade de dados e à apresentação de reclamações junto a autoridades de controle.

Informações para Usuários nos Estados Unidos (CCPA / CPRA)

Para usuários residentes na Califórnia ou outros estados dos EUA com legislações de privacidade, aplicam-se direitos adicionais conforme o CCPA/CPRA:

  • Direito de saber quais dados são coletados
  • Direito de solicitar a exclusão dos dados pessoais
  • Direito de se opor à "venda ou compartilhamento" de dados pessoais
  • Direito ao uso não discriminatório do aplicativo

O opt-out pode ser feito de duas formas:

  1. Configurações do iOS: Vá em Ajustes > Privacidade e Segurança > Rastreamento para desativar o rastreamento.
  2. Dentro do aplicativo: O aplicativo transmite automaticamente sinais de privacidade (como RDP ou IAB Privacy String) para refletir a escolha do usuário.

Mais informações sobre o tratamento de dados pela Google e opções de exclusão:
https://policies.google.com/privacy

As categorias de dados processados incluem, entre outras:

  • Identificadores (por exemplo, ID do dispositivo, endereço IP)
  • Informações de uso (por exemplo, atividade no app, interações com anúncios)
  • Dados de localização aproximada
  • Informações derivadas (por exemplo, perfis de interesse)

Informações para Usuários no Brasil (LGPD)

Este aplicativo está em conformidade com a Lei Geral de Proteção de Dados Pessoais (Lei nº 13.709/2018).

Os dados pessoais só são tratados com base em uma permissão legal – em especial com consentimento explícito ou interesse legítimo.

De acordo com a LGPD, os usuários têm os seguintes direitos:

  • Acesso aos dados pessoais armazenados
  • Correção ou atualização dos dados
  • Exclusão ou bloqueio dos dados
  • Revogação do consentimento
  • Restrição ou oposição ao tratamento de dados

Para exercer esses direitos, entre em contato com o controlador através do e-mail indicado acima.
